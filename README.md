# Global-Economic-Inequality-Poverty-Analysis-1980-2024

This project analyzes global inequality, poverty, and income distribution trends using country-level data.

## Features

* Inequality (Gini) and poverty trends
* Income gap (Top 10% vs Bottom 50%)
* Country comparison
* Global inequality map
* Interactive dashboard

## Tech Stack

Python, Pandas, Plotly, Streamlit

## Run

```bash
streamlit run app.py
```
